# AnkiThemeTwin Config

- `currentTheme`: The selected theme. One of `"sepia_word"`, `"sepia_paper"`, `"sepia_special"`, `"gray_word"`, `"gray_paper"`, `"blue_light"`, or `"olive_green"`.
- `fontSize`: Font size in pixels for webview content. Options: 14, 16 (default), 18, or 20.
- `followSystemTheme`: When `true`, addon theming is disabled and Anki's native dark/light mode controls the appearance. Default: `false`.
