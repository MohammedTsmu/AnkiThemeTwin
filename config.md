# AnkiThemeTwin Config

- `currentTheme`: The manually selected theme. One of `"sepia_word"`, `"sepia_paper"`, `"sepia_special"`, `"gray_word"`, `"gray_paper"`, `"blue_light"`, `"olive_green"`, `"dark_warm_soft"`, `"dark_neutral_soft"`, `"true_black"`.
- `followSystem`: `true` to automatically switch between light and dark themes based on the OS setting, `false` to use `currentTheme`.
- `lightTheme`: Theme to use when the OS is in **light mode** (only applies when `followSystem` is `true`).
- `darkTheme`: Theme to use when the OS is in **dark mode** (only applies when `followSystem` is `true`).
- `fontSize`: Font size in pixels for webview content. Options: 14, 16 (default), 18, or 20.
